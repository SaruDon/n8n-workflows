# n8n Workflows Repository

A collection of reusable n8n workflows for automation projects.

## Structure

workflows/
  Each workflow has its own folder containing:
  - workflow.json
  - README.md
  - meta.json

shared/
  Shared scripts, examples, and utilities.

## How To Use

1. Open your n8n instance
2. Import workflow.json
3. Follow README inside workflow folder
